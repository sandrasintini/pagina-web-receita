Esse projeto é uma página web estática desenvolvida como projeto de aprendizagem, com foco na criação de um layout para apresentação de receitas.

O projeto foi pensado para oferecer uma experiência de leitura confortável, valorizando o conteúdo e a estética da página, como uma folha de receita digital.

O objetivo aqui é praticar HTML semântico, utilizar CSS puro, sem dependências externas e demonstrar atenção a detalhes de estrutura, estilo e experiência do usuário.

Técnologias utilizadas: 
  - HTML5
  - CSS3
    
Conceitos aplicados:
  - Estrutura semântica clara (header, main, section, footer);
  - Hierarquia visual e tipografia voltadas para leitura;
  - Separação entre conteúdo e estilo;
  - Design simples, funcional e orientado à experiencia do usuário
